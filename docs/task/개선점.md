## 현재 정리 기준

- 리워드 기능은 계속 미완 상태로 유지합니다.
- 이유는 기획과 운영 정책이 아직 충분히 정리되지 않았기 때문입니다.
- 따라서 리워드 관련 항목은 `즉시 개발 대상`이 아니라 `보류/추후 기획 대상`으로 분리해 관리합니다.
