# 2026 디자인 트렌드

## 핵심 트렌드 5개

1. Quiet Productivity UI
2. Proof-Driven Product Pages
3. Community-As-Workflow
4. Personal Reputation Layer
5. Institution-Ready Platforms

## 이 서비스에 맞는 방향

- 채택:
  - 조용한 인터페이스
  - 상태 중심 UX
  - 활동 증거 기반 신뢰 요소
  - 공개 운영 타임라인
- 피할 것:
  - 과도한 소셜 피드화
  - 의미 없는 게임화
  - 미완성 기능의 과노출

## 경쟁 구도 인상

| 레퍼런스 축 | 가져올 점 | 피할 점 |
|---|---|---|
| Product Hunt | 신작 발견감 | 하루성 소비 구조 |
| GitHub Issues | 구조화된 피드백 | 너무 개발자 중심 용어 |
| Notion형 포트폴리오 | 정리된 소개 | 너무 정적임 |
| Discord형 커뮤니티 | 친밀한 소통 | 정보 분산 |
| Behance형 전시 | 쇼케이스 감성 | 결과물만 강조 |

## 사용자 기대 변화

- 사용자는 결과물보다 활동성과 실행력을 봅니다.
- 교육기관은 단순 나열보다 성장 데이터가 있는 포트폴리오를 원합니다.
- 메이커는 비개발자도 이해할 수 있는 운영 레이어를 필요로 합니다.
